TaskFlow AI is a premium full-stack AI-powered task management platform built using React, Vite, Spring Boot, MySQL, and WebSockets. Inspired by Linear and Vercel, it features a futuristic glassmorphism UI, real-time collaborative Kanban boards, AI-powered productivity insights, JWT authentication, STOMP-based live synchronization, and interactive analytics dashboards using Recharts. The platform supports intelligent task decomposition, sprint health analysis, drag-and-drop workflow management, and responsive cross-device collaboration with a modern SaaS-grade user experience.
